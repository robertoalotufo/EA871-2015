Introdução

Esta página contém a documentação dos oferecimentos do curso EA871 para o primeiro e o segundo semestre de 2015. Oferecida semestralmente, a disciplina faz parte do currículo dos cursos da Faculdade de Engenharia Elétrica e de Computação da UNICAMP.

O objetivo dessa disciplina é ensinar a programação de microcontroladores e explorar os conceitos abordados nas disciplinas teóricas de computação, como arquitetura de computadores, linguagem de montagem, entrada/saída e interrupções.

Ementa

  • Programação em linguagem Assembly.
  • Instruções de máquina.
  • Diretivas do montador.
  • Depuradores.
  • Interfaces de entrada/saída.
  • Programação de interface serial.
  • Comunicação entre computadores.
  • Programação de interface paralela.
  • Interrupções.

Pré-requisitos

MC404 - Organização Básica de Computadores e Linguagem de Montagem (Engenharia de Computação)

  • Carga total : 60 horas (4 créditos)
